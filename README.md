## 📌 Overview

This is a Unity multiplayer tank battle game where players compete to collect coins and outscore their opponents within a limited time.

The project supports:

- Offline mode (local play)
- Online multiplayer mode
- Tank selection system
- Skill-based gameplay
- Resource-based ability system
- Timed match win condition

The main focus of this project is combining real-time combat with strategic resource management.

---

## 🕹 Game Modes

### 📴 Offline Mode
- Players can play locally.
- Practice movement, shooting and skill usage.

### 🌐 Online Mode
- Players connect and compete in real-time.
- Score is synchronized between players.
- Match ends after a fixed time.

---

## 🎮 Core Gameplay

### 🚜 Tank Selection
Before entering the match, players can select their tank.

Each tank has:

- Movement
- Shooting ability
- 3 special skills

---

## ⚡ Tank Skills

Players can activate skills using keyboard inputs:

- **Press 1 → Speed Boost**
  - Temporarily increases movement speed

- **Press 2 → Rapid Fire**
  - Increases shooting speed

- **Press 3 → Heal**
  - Restores HP

---

## 💰 Coin System

Coins are scattered around the map.

Players can:
- Collect coins by moving over them
- Spend coins to:
  - Shoot bullets
  - Activate skills

This creates a strategic layer:
- Use coins for combat?
- Or save coins to win by score?

---

## 🏆 Win Condition

- The match runs for a fixed duration.
- The player who collects the most coins when time ends wins.

Victory depends on:
- Efficient coin collection
- Smart skill usage
- Combat strategy
